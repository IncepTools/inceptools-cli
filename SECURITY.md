# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| latest  | ✅ Yes             |
| < latest| ❌ No              |

## Reporting a Vulnerability

If you discover a security vulnerability in **inceptools**, please report it responsibly:

1. **Do not** open a public GitHub issue
2. Use [GitHub's private vulnerability reporting](https://github.com/IncepTools/inceptools-cli/security/advisories/new) or email the maintainers with:
   - A description of the vulnerability
   - Steps to reproduce
   - Potential impact
3. You will receive a response within **72 hours**

We appreciate your help in keeping inceptools secure.
